🌐 Sphere Viewer - Embeddable Component Demo

Demonstrate how to embed the Sphere Viewer into any webpage

📋 How to Use

  1. Build the embeddable component: npm run build:embed
  2. Include React and ReactDOM from CDN (or your preferred method)
  3. Include the sphere-viewer.js script with your session ID
  4. The component will automatically initialize!

Method 1: Auto-initialization with data attributes

The simplest way - just include the script tag with data attributes:

<script src="dist/sphere-viewer.js" data-session-id="your-session-id-here" data-container-id="my-sphere-container"></script> <div id="my-sphere-container"></div>

Auto-initialization demo
(Replace with actual session ID to test)

Method 2: Manual initialization with JavaScript API

For more control, use the JavaScript API:

<script src="dist/sphere-viewer.js"></script> <script> const viewer = new window.SphereViewer(); viewer.init({ sessionId: 'your-session-id-here', containerId: 'my-custom-container', apiBaseUrl: 'https://sphere-api.featrix.com' // optional }); </script> <div id="my-custom-container"></div>

Manual initialization demo
(Use browser console to test API)

Method 3: Dynamic Updates

Update the viewer with new session data:

// Update to a new session viewer.update({ sessionId: 'new-session-id' }); // Destroy the viewer viewer.destroy();

🛠️ Build Instructions

  1. Install dependencies: npm install
  2. Build the embeddable component: npm run build:embed
  3. The output will be in dist/sphere-viewer.js
  4. Include this file in your webpage along with React dependencies
