💡 Customization Options:

  • data-session-id - Your Featrix session ID
  • data-is-rotating - Enable/disable sphere rotation (true/false)
  • data-rotation-speed - Rotation speed (0.01 to 0.5)
  • data-animate-clusters - Auto-cycle through clusters (true/false)
  • data-point-size - Size of data points (0.01 to 0.2)
  • data-point-opacity - Transparency of points (0.1 to 1.0)
  • data-container-id - Target container ID (optional)
  • data-api-base-url - Custom API endpoint (optional)
